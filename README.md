# Zameal Client

The web client of the Zameal Tarheel board, built with NextJs, typescript and tailwindcss.

## Demo

https://zameal.vercel.app

## Deployment

To deploy this project run

```bash
  npm run start
```

or

```bash
  docker build -t web
  docker run -p 3000:3000 web
```
